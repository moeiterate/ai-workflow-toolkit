# Goal Definition

**When to use:** Starting a new project. First of 3-conversation validation.

## Prompt

```
I want to build [CONCEPT]. Help me define:

1. The core problem this solves
2. Who the primary users are
3. What success looks like (measurable metrics)
4. Key technical requirements or constraints
5. Budget and timeline considerations

Be specific and challenge vague requirements.
```

## Usage

- Use in dedicated AI conversation
- Replace [CONCEPT] with your idea
- Be specific about constraints
- Save output as PROJECT_GOALS.md
